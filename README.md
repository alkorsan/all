## Installation

#### Node.js

```bash
npm install lil.js
```

#### Browser

Via [Bower](http://bower.io)
```bash
bower install lil
```
Via [Component](https://github.com/componentjs/component)
```bash
component install lil-js/http
```

Or loading the script remotely
```html
<script src="//cdn.rawgit.com/lil-js/all/0.1.0/lil.js"></script>
```
